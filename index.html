    Frequently Asked Questions (FAQ)

    What is Pomodoro Technique?

    The time management technique created by Francesco Cirillo for a more productive way to work and study. For more information, click here.

    Can you tell me the story without having to visit the website?

    Well, it comprises of the following basic steps:

    1. Decide on the task at hand
    2. Set the Pomodoro (timer) to 25 minutes
    3. Work on the task until the timer expires; Record with an X
    4. Take a Short Break (5 minutes)
    5. Every four "pomodoros", take a Long Break (10 minutes)

    What is TomatoTimer?

    It's an easy to use, flexible Pomodoro Technique timer. This project is a clone of tomato-timer.com (now toptal.com/project-managers/tomato-timer) and it uses Vanilla JavaScript and HTML5 features like Desktop Notifications, Audio API and Local Storage instead of relying on Adobe Flash and other such technologies.

    Why use TomatoTimer?

    Here's why:

    • Clean and Crisp interface with a Mobile friendly layout.
    • Ability to Pause or Reset the timer intervals.
    • Audio notifications at the end of a timer period.
    • Desktop notifications.
    • Keyboard shortcuts.
    • Ability to change the alert sound + volume via Settings
    • Custom Timer Intervals
    • A log of your activity.
